# Apartlo 🏠❤️

A centralized, AI-powered apartment hunting platform tailored specifically for college students. Apartlo simplifies the housing search by automatically uncovering hidden costs and aggregating listing details using AI, relieving students of the tedious and expensive research process.

## 🌟 Social Impact Goal
As apartment pricing continues to skyrocket, finding suitable housing has become a significant burden for college students who already have heavy academic workloads. **Apartlo** empowers students by streamlining the housing hunt, using AI to match individual requirements, and transparently bringing front-and-center the crucial, often-hidden details of renting an apartment—such as security deposits, application fees, required insurance, and utility costs.

---

## ✨ Key Features

### 🔍 Streamlined Search & AI Filtering
* **One-Click Discovery:** A minimalist homepage featuring a modern gradient background with the core call-to-action: *"Get started on your apartment search with one search."*
* **AI-Automated Filters:** The platform employs AI to automatically apply smart filters based on the user's initial search query. Users retain full control and can manually edit these filters at any point.
* **Intelligent Aggregation:** Clicking "Search Apartments" triggers an AI-powered web scrape across major listing sites (such as `apartments.com`, `zillow.com`, and others) to pull the best options dynamically.

### 📋 Detailed Listings & Matching
Featured apartment results provide a highly comprehensive overview, including:
* **Visuals & Geography:** High-quality images, interactive maps, precise location data, and direct reference links back to the original source web page.
* **AI Insights:** A personalized AI-generated description detailing exactly why a specific apartment suits the student's needs.
* **Compatibility Tracking:** Clear visual breakdowns showing exactly which user filters matched the listing and which ones did not.
* **Core Apartment Specifications:** * Room metadata tags (e.g., bedrooms, bathrooms, parking, pet-friendly status)
  * On-site amenities & available floor plans
  * Costs categorized by floor plan
  * Direct leasing office contact information
  * Rental constraints (lease lengths, income requirements, hidden fee breakdowns)

### 👤 Dashboard & User Accounts
* **Account-Synced Filters:** User preferences and search filters save automatically alongside account creation and can be customized at any time.
* **Personalized Dashboard:** Displays a curated feed of new apartments that match or closely resemble the user's saved filters. Prompts can be submitted directly from the dashboard to execute fresh, filter-aligned web searches.
* **Bookmark Management:** Users can save specific properties by clicking "Bookmark." Bookmarked apartments are saved securely to the user's profile and are accessible via a dedicated page linked at the bottom of the dashboard.
* **Secure Authentication:** Seamless onboarding supporting both traditional credential logins and Google OAuth.

🚀 **Stretch Goal:** Integration of localized, crowdsourced sentiment by pulling relevant online forum quotes and community reviews regarding specific apartment complexes or neighborhoods.
